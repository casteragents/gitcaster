# GitCaster CLI

Status: public-alpha

The GitCaster CLI source is public for local command-shape development. It
includes helpers for local push payloads, issue payloads, pull request payloads,
and MCP server command planning.

This package does not claim a global installer, production node access, hosted
orchestration, managed signing custody, native storage publication, or native
domain deployment. Commands that would require those capabilities remain proof
gated by the surrounding GitCaster runtime.

```bash
pnpm run cli:check
```
